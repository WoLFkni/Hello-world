## 无人称结构
1. It is +p.p. + that 
2. It is + adj. + that clause
3. It is  noun + that clause
### It is + p.p. (过去分词)+ that clause

	It is quoted as saying that ...
	It is reported that

> 人们希望
	It is anticipated that 
	It is hoped that 
	It is expected that

expected sth. of sb. 注意人和物的顺序 期待某人干某事

> 有人提到
	It is mentioned that

> 据预测
